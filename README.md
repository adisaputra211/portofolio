## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.js`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## 📌Portfolio Website – Adi Saputra
A modern and responsive personal portfolio website built using Next.js and React, designed to showcase projects, technical skills, and professional experience in a clean and structured way.

This project emphasizes performance optimization, modular architecture, and scalable frontend development practices.

## 🚀 Tech Stack
- Next.js (App Router)
- React
- CSS
- Intersection Observer API (Scroll Reveal Animation)
- Vercel Deployment
  
## ✨ Key Features
- Deployed on Vercel for production stability
- Deployed on Vercel for production stability
- Fast and optimized performance using Next.js
- Modular component structure
- Fully responsive design (mobile-first)

## 🏗 Architecture Approach
The application follows a modular component-based structure to ensure:
- Clean code organization
- Easy maintenance
- Scalability for future feature expansion
Animations are implemented using the native Intersection Observer API to maintain performance efficiency without heavy animation libraries.
